Markov chains based on words

Use preloaded text?

This is sketch uses Markov chains to predict the next word based on n previous words. The seed text is divided in groups of n words in order they follow each other and records the sequence into a table with first n-1 words as key and the last as a value in an array. Then, it generates text by picking random words based on n-1 previous ones many times.
