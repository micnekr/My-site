Neural network XOR problem

This sketch is solving XOR problem using neural networks. The input varies from 0,0 at the top left corner and 1,1 at the bottom right corner. The colour of each cell is the output of the neural network mapped to a colour between black for false and white for true. It is impossible to solve this problem with single neuron because can not be solved by drawing one line (what neuron does) to separate all zones, so a neural network is used.
