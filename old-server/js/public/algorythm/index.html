Maze constructor and solver

This sketch shows two algorithms: recursive backtracker for building mazes and A* pathfinding algorithm. At first, recursive backtracker algorithm is used to build a mase. Then, A* algorithm is used to solve this maze. Green cells represent open set in A* and current cell in backtracker. Blue cells represent cells visited by backtracker and blue line in the current best path found by A* algorithm. Black lines are walls and white cells are unvisited cells.
